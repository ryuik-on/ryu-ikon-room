# Visual direction — discussion draft

これは今後Claude・人間と検討するための前提であり、決定済みデザインではありません。

## 確認済みの方向

- 写真、映像、余白を主役にする
- 奏者と観客の間に生まれる空間を表現する
- 派手さだけでなく、緊張感や人の熱を感じる
- 学園祭の告知サイトに見せない
- テンプレート的なロックサイトに見せない
- モバイルを最優先する
- 説明より先に体験を置く

## 避けるもの

- 旧サイトの見た目の踏襲
- ネオン調
- 音符や波形の装飾
- 大量の英語コピー
- 黒背景に巨大文字を置くだけの構成
- 過剰なスクロールアニメーション
- 不要なグラデーション
- テンプレート的なバンドサイト
- 装飾を目的とした演出

## ROOMのコンセプト原文

> 奏者が、それぞれの日常で
>
> 「これでいいんだろうか」と迷いながらも必死に音を鳴らし、
>
> その熱を、必要としている人へまっすぐ届ける。

この文章は要約・言い換えをせず、原文として扱います。

## 今後の検討軸（提案であり未決定）

- 写真の選定は「格好よさ」だけでなく、奏者と観客の距離、迷い、集中、熱が写っているかで評価する
- 動きは意味のある場面に限定し、静止状態でも内容が成立することを先に確認する
- 余白は装飾ではなく、視線と感情の切り替えに使う
- 色と書体は素材の色・主催者の既存ブランド・可読性を確認してから決定する
- Heroは最初の5秒で公演名と日程を読めることを優先し、演出で遅らせない
